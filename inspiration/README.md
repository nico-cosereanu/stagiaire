# Design inspiration

Drop screenshots, mood images, and reference URLs in this folder. Both Claude Code and I (in chat) can read them. The more specific you are about *what* you like in each ref, the better the resulting design.

## How to add a reference

### Screenshots
Drop the image into the appropriate subfolder. **Filename matters** — it's how I'll refer to it. Format:

```
[source]_[what-it-shows].png
```

Examples:
- `apple-fitness_globe-rotation.png`
- `apartamento_serif-headline.png`
- `airbnb_listing-card-mobile.png`
- `stripe-atlas_form-density.png`

### URLs / Pinterest boards / Are.na channels / Figma files

Add them to `LINKS.md` in this folder with a one-line note on what to look at.

### Notes per ref

If you want to be specific about *what* you like in a ref (not the whole image), add a sibling `.md` file with the same base name:

```
apartamento_serif-headline.png
apartamento_serif-headline.md   ← "headline weight + line-height, ignore the photo"
```

Optional but very useful.

## Subfolders — what goes where

- **`globe/`** — anything globe-related: 3D earths, map projections, pin styles, dot maps, line-art globes
- **`typography/`** — font pairings, scale, weight, hierarchy, editorial layouts
- **`layout/`** — page structure, card design, dashboards, profile pages, grid systems
- **`color/`** — palettes, monochrome studies, accent usage
- **`photography/`** — kitchen/food/portrait reference, image treatment, photo cropping
- **`motion/`** — page transitions, micro-interactions (link to videos in LINKS.md if not a static frame)
- **`full-products/`** — products whose overall vibe you want to channel (Linear, Stripe, Things, Are.na, etc.)
- **`competitors/`** — CulinaryAgents, Poached, StarChefs, anything in the same space — what to do *and what not to do*

## What I'm looking for from you

A handful of refs across categories beats a hundred screenshots in one bucket. If you can give me even three or four images per category with one-line notes, I have plenty to work with.

The single most useful thing you can give me: **two or three full-product refs** whose overall feel you want Stagiaire to land near. (e.g., "I want the app to feel like Linear meets Are.na meets the Apartamento website.")
